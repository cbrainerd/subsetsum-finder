# subsetsum-finder

Find all unique combinations in the input list that sum up to the target.

## Development

### Setup

Install uv: https://docs.astral.sh/uv/getting-started/installation/

```
uv sync
```

### Lint

```
uv run ruff check
```

### Code Formatting

```
uv run ruff format
```

### Unit Tests

```
uv run pytest tests/
```
